# BCG Careers Section Design Notes

## Key Design Elements Observed:

### Hero Section
- Green background with bold white headline "Beyond is where we begin."
- Search bar prominently placed
- Image card with caption showing team member in office
- Clean, minimal layout

### Career Paths Section
- Three cards in a row: Students, Digital/Technology, Experienced Professionals
- Each card has:
  - Full-bleed image with person
  - Caption showing name and role
  - Category label (uppercase, small)
  - Brief description
  - "LEARN MORE >" link with arrow

### Locations Section
- Two-column layout
- Left: Stacked polaroid-style images
- Right: Heading + description + green CTA button
- Clean white background

### Social Feed Section
- "Get a glimpse into life at BCG" heading
- Horizontal carousel of social posts
- #BCGLIFE branding

## Design Principles for NexDyne Careers Section:
1. Use asymmetric layout (text left, image right or vice versa)
2. Clean typography with clear hierarchy
3. Professional imagery showing diverse team
4. Statistics/numbers with clear labels
5. Clear CTA with arrow
6. Minimal, clean aesthetic
7. Two-column or card-based layouts
