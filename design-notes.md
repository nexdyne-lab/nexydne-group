# Bain-Style Homepage Design Notes (January 2026)

## Typography
- **Headlines**: Georgia (serif) - large, bold, elegant
- **Body text**: Helvetica Neue or similar sans-serif
- **Navigation**: Sans-serif, uppercase for utility links

## Color Palette
- **Primary Red**: #CC0000 (Bain red)
- **Dark backgrounds**: Deep maroon/burgundy gradients, dark images
- **White text on dark backgrounds
- **Red accent indicator for active states

## Hero Layout
- Full-screen background image (100vh minus navbar)
- Category label above headline (smaller, white, uppercase)
- Large serif headline (left-aligned, max-width ~50%)
- "READ MORE" CTA with arrow
- Bottom navigation tabs showing 4 featured items
- Active tab has red underline indicator
- Smooth crossfade transitions between slides

## Navigation Structure
- Top utility bar (offices, alumni, media, subscribe, contact)
- Main nav with logo + dropdown menus
- Search and bookmark icons on right
- Hamburger menu on left

## Featured Content Tabs (Bottom)
- 4 tabs evenly spaced
- Gray text for inactive, white for active
- Red underline indicator below active tab
- Clicking tab changes hero background and content
