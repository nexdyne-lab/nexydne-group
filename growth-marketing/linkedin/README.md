# LinkedIn — single source of truth

Everything LinkedIn for NexDyne, in one place.

- **`LINKEDIN_STRATEGY.md`** — start here. Founder-led strategy, posting mechanics, content mix, render pipeline, backlog.
- **`linkedin-posts.md`** — the master content plan: full copy for every post + publish status + coaching notes.
- **`LinkedIn Content Calendar.xlsx`** — the tracker (date, post #, type, asset, status).
- **`2026-07/`, `2026-08/`** — monthly visual assets, named by post date; each has a `POSTS.md` index.
- **`assets/page/`** — LinkedIn Page identity (logo 300×300, banner 1128×191).
- **`assets/hero-backgrounds/`** — 4 custom iso line-art backgrounds (HIG/AI/Ops/Cloud).
- **`assets/extra-visuals/`** — milestone/hiring one-off post images.

Render machinery (HTML + Puppeteer) lives in `../../../social-tiles/` — see the
strategy doc's "Render pipeline" section.
